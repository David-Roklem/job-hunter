# Vision

`job_hunter` — персональный локальный помощник поиска работы на **международном
и зарубежном рынке** (с сохранением российского рынка через hh.ru). Он
автоматически собирает вакансии из hh.ru, зарубежных площадок (Wellfound и др.),
сайтов компаний и Telegram-каналов, сопоставляет их с несколькими версиями резюме,
готовит черновики откликов и сопроводительных писем через AI — а человек только
подтверждает.

Режим работы: **«система готовит → ты одобряешь → система отправляет»**.
Без хостинга, биллинга, регистрации и публичного API — это single-user инструмент,
запускаемый локально и оптимизированный под среднюю нагрузку ~100 откликов/день.

## Goals

- Автоматический сбор вакансий из нескольких источников (hh.ru, Wellfound,
  карьерные страницы компаний, Telegram-каналы) через Playwright с анти-лимитами
  и троттлингом.
- Несколько резюме-шаблонов под разные роли/направления; матчинг вакансия↔шаблон
  по релевантности навыков с скорингом кандидатов.
- AI-генерация адаптированного резюме и сопроводительного письма под конкретную
  вакансию (мультиязычные промпты — язык параметризуется источником/локалью;
  российский рынок через hh.ru остаётся поддержанным).
- Инбокс «подтвердить / редактировать / отклонить» — подготовленные отклики
  одобряются одним кликом, после чего авто-отклик выполняется на hh.ru.
- Фоновый планировщик с очередью задач: регулярный сбор, троттлинг запросов,
  логирование — устойчиво держит целевой объём откликов в день.

## Источники и рынки

- **Российский рынок:** hh.ru (Playwright + анти-детект).
- **Международный/зарубежный рынок:** Wellfound и другие агреграторы
  (адаптивные селекторы, Playwright; ручной логин по необходимости).

## Non-goals

- Публичный хостинг, многопользовательский режим, биллинг, регистрация аккаунтов
  — проект строго single-user и локальный.
- Полный автопилот без участия человека — каждое отклонение подтверждается;
  безопасность и конверсия важнее сырного объёма.
- Мобильные клиенты, нативные приложения, веб-доступ извне сети машины.
- Сбор вакансий только через официальный API hh.ru — отказались из-за
  OAuth-бюрократии; используем браузерную автоматизацию Playwright.

## История изменений

- **2026-07-13:** Фокус смещён с «только РФ» на «международный/зарубежный рынок
  + hh.ru». Добавлен Wellfound как первый зарубежный источник. Язык промптов
  параметризуется локалью (раньше — только русский, заложена мультиязычность).
